8. 以前，3の倍数と3が付く数値のときだけアホになるというギャグがあった．

ただし作成するプログラムにおいて，「アホ」という言葉は教育上ふさわしくない．
そこで3の倍数と3が付く数値のときは「いまでしょ!」と出力し，それ以外の数値では
