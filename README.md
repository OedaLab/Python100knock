27. あみだくじを実行するプログラムを作成せよ．

0,1,0は2本目と3本目が第2段階でつながっていることを示している．
このデータ場合，1, 2, 3, 4のくじを選ぶと，どこに到達するか計算する
プログラムを作成する．ちなみにこのデータでは，それぞれ3, 1, 2, 4に到達する．

[データ](./amida.csv)
